# Workloom Studio

Dark theme landing page plus login and signup with a custom Node.js auth API.

## Frontend

1. npm install
2. npm run dev

Create a .env file based on .env.example if you need a custom API URL.

## Backend

1. cd server
2. npm install
3. Create a .env file based on server/.env.example
4. npm run dev

The API runs on localhost port 4000 and expects MongoDB to be available.

## Auth flows included

- Email and password sign up
- Email verification
- Login with secure cookies
- Password reset
